# **CashFlow**

CashFlow é uma API REST criada para ajudar no acompanhamento das suas despesas. Com ela, você consegue controlar seus gastos de forma organizada e ainda gerar relatórios em PDF ou Excel, facilitando o acompanhamento e a análise das suas finanças.
